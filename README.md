# 🚤 Tretboot – Konstruktionsprojekt

## 📌 Projektbeschreibung

Dieses Projekt entstand im Rahmen einer universitären Aufgabe im Modul **Konstruktion**. Ziel war die Entwicklung und Konstruktion eines funktionsfähigen **Tretboots** mithilfe moderner CAD-Software.

---

## 🎯 Zielsetzung

* Entwicklung eines vollständigen 3D-CAD-Modells eines Tretboots
* Konstruktion aller relevanten Baugruppen
* Berücksichtigung von:

  * Funktionalität
  * Stabilität
  * Fertigbarkeit
  * Montage

---

## 🛠️ Inhalte des Projekts

Das Repository enthält:

* 📐 **CAD-Modelle** (Einzelteile & Baugruppen)
* 🧩 **Zusammenbau (Assembly)** des Tretboots
* 📄 **Technische Zeichnungen**
* ⚙️ Konstruktive Lösungen für:

  * Antriebssystem (Pedale → Vortrieb)
  * Rumpf / Schwimmkörper
  * Lenkmechanismus

---

## ⚙️ Funktionsprinzip

Das Tretboot basiert auf einem klassischen mechanischen Antrieb:

1. Fahrer tritt Pedale
2. Kraftübertragung über Kurbel und Antriebssystem
3. Umsetzung der Energie in Vortrieb (z. B. Propeller oder Schaufelrad)
4. Steuerung über Rudermechanismus

👉 Die Antriebskraft wird direkt durch Muskelkraft erzeugt und mechanisch weitergeleitet ([Google Patente][2])

---

## 🧠 Konstruktive Schwerpunkte

* 💡 **Leichtbauweise** zur Effizienzsteigerung
* ⚖️ **Stabilität im Wasser** (z. B. Katamaran-Prinzip)
* 🔧 **Modularer Aufbau** für einfache Montage
* 🔄 **Kraftübertragung** mit möglichst geringem Verlust
